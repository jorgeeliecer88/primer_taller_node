
Si se trabaja proyecto comprimido:

Descomprimir el zip: primer_taller
ejecutar en linea de comandos primera parte : node principal
ejecutar en linea de comandos segunda parte : node principal inscribir --i=1 --n=jorge --x=12344

Si se trabaja con proyecto github:

Descargar proyecto:
https://github.com/jorgeeliecer88/primer_taller_node

Ejecutar dentro del proyecto el comando install para instalar descargar dependencias: 
npm install

Ejecutar en linea de comandos primera parte : node principal

Ejecutar en linea de comandos segunda parte : node principal inscribir --i=1 --n=jorge --x=12344
